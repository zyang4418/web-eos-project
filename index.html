Frontend Essentials

Explore the core technologies, tools, and principles that power modern web experiences.

HTML5

HTML5 is the modern standard for structuring content on the web, introducing semantic elements that make pages more meaningful and accessible. It also includes built-in support for multimedia, graphics, and offline capabilities.

CSS

CSS defines how HTML elements are styled and displayed, controlling layout, colors, typography, and overall visual design. It separates structure from presentation, making websites easier to maintain and customize.

Google Lighthouse

Google Lighthouse is an automated auditing tool that evaluates web page performance, accessibility, SEO, and best practices. It provides actionable insights to help developers optimize their sites for quality and speed.

JavaScript

JavaScript is a versatile programming language that adds interactivity and dynamic behavior to web pages. It runs in the browser and powers everything from animations to complex web applications.

Cookies and localStorage

Cookies and localStorage are client-side storage mechanisms used to save data in the browser. Cookies are typically used for session management and expire over time, while localStorage provides persistent key-value storage for larger amounts of data.

HCI (Human‚ÄìComputer Interaction)

HCI studies how people interact with computers and focuses on designing interfaces that are usable, efficient, and intuitive. It combines psychology, design, and technology to improve user experience.

jQuery

jQuery is a lightweight JavaScript library that simplifies DOM manipulation, event handling, and AJAX operations. It offers a concise syntax that works consistently across different browsers.

jQuery UI

jQuery UI is a collection of user interface components built on top of jQuery. It provides ready-made widgets like dialogs, sliders, and date pickers, along with theming and interaction effects.

jQuery Validation

jQuery Validation is a plugin designed to simplify client-side form validation. It offers customizable rules and messages, making it easy to ensure user input meets required criteria before submission.
